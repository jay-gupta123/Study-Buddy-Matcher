# Studdy Buddy

Studdy Buddy is a web-based study assistance platform designed to help students improve productivity, organize learning resources, and enhance their study experience through an interactive and user-friendly interface.

---

## 🚀 Features

* User-friendly and responsive interface
* Interactive study tools and utilities
* Dynamic frontend functionality using JavaScript
* Python backend support for logic/processing
* Clean and organized design for better user experience

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Python

---

## 📂 Project Structure

```
Studdy-Buddy/
│── index.html
│── style.css
│── script.js
│── app.py
│── assets/
└── README.md
```

---

## ⚙️ Installation & Setup

### 1. Clone the Repository

```
git clone https://study-buddy-matcher.vercel.app/
```

### 2. Navigate to Project Folder

```
cd studdy-buddy
```

### 3. Run Python Backend

```
python app.py
```
